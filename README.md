# Stormpal: Your Go-To Weather CLI Tool
Stormpal is the perfect companion for anyone looking to stay up-to-date with the latest weather forecasts from the comfort of their terminal.

## Description
Stormpal is a Python-based weather CLI tool designed to provide users with current weather conditions, forecasts, and other relevant weather-related information. With its simple and intuitive interface, you can easily access the weather data you need without having to navigate through complex websites or apps.

## Features
* **Current Weather Conditions**: Get the current temperature, humidity, wind speed, and other weather conditions for any location.
* **Weather Forecasts**: Stay ahead of the weather with accurate forecasts for the next few days.
* **Location Support**: Stormpal allows you to retrieve weather data for any location, whether it's your current city or a destination you're planning to visit.
* **Customizable**: With Stormpal, you can customize the units of measurement and other settings to suit your preferences.

## Installation and Usage
To get started with Stormpal, follow these simple steps:
1. **Install dependencies**: Run `pip install -r requirements.txt` in your terminal to install the required dependencies.
2. **Run**: Execute `python main.py` to launch the Stormpal CLI tool.
